# 终局质量指引
## 开篇承诺回看
这部书开头抓人的，不只是“坠轨命案”，而是**不存在的到站记录**，以及“真证据彼此冲突”的系统性异常；情感钩子则是那句**“说明你又把第零天忘了。”** 所以终局必须回应的，不是“制度终于修好了”，而是：**当现实承认缺口后，人还要怎么活；当真相会切掉自我时，陆既白最后选择保住什么。**  
你现在已经把“规则落锁”写得很稳了，接下来最后1~2章不能继续证明制度正确，而要让读者真正看见：**这套代价，有人接住了；这份空白，被现实承认了；陆既白没有把自己再写回去。**

## 当前终局最容易失手的地方
最大风险，是把后两章写成“新规运行汇报 + 社会接受说明 + 各方立场补丁”。现在该解释的已基本够了，再写多就会变成**制度说明会 / 办结通知**。  
第二个风险，是重复“留空合法”这个结论。第231—234章已经反复验证过，后面不能再来一轮同构个案证明。**要从‘规则成立’转去‘人如何承受’。**  
第三个风险，是情绪一路往下收成疲惫、克制、沉静，最后只剩“都结束了”。这会让当前很好的落锁感，变成尾声无波。最后必须重新抬起的，是：**陆既白的选择重量、陆南枝留下的回声、空白第一次被普通人真正接住时的刺痛与平静。**

## 必须打中的主题回答
最后终局要明确回答三件事：  
一，**真相的终点不是补完，而是停止伪造。** 这句主题你已写出，但还需要在现实层被一个具体人接受，而不是只停在馆内屏幕。  
二，**陆既白不是因为无力才放弃代写，而是主动拒绝成为历史补位来源。** 这是他对开篇“反复遗忘、逼近自己参与过的秘密”的最终回答：他不再用自己去填那个秘密。  
三，**陆南枝留下的不是答案，而是边界。** 她不能在结尾再提供新信息；她的作用应是让陆既白最后那一步带有亲情回声——他记不全她，但他守住了她想守的东西。  
如果当前结尾偏解释化，最后1章要重新抬起来的，不是机制细节，而是**“有人拿到留空回执时，终于不再逼系统撒谎”**；最后2章结构则应是：一章落现实，一章留余韵。

## 最后一屏建议
最后一屏不要是会后批注，不要是制度通告，也不要是又一份归档说明。  
最好是一个极具体、可停留的画面：**普通窗口/家属/当事人拿着那份“有效但留空”的正式回执离开，停顿片刻，没有再追问系统替谁补名；陆既白在稍远处看见这一幕，没有上前解释，只确认原始链封存完毕。**  
然后接一个极短的私人回声：比如夜里、站台、玻璃反光、旧录音标题、或一条再普通不过的进站提示音，让读者想起最初“根本不存在的到站记录”。但这一次，不是逼它出现，而是**允许它不存在**。  
最后一段应给读者的感觉是：**不是胜利，不是总结，而是一种迟来的、带代价的平静——这座城终于停止补字，这个人终于停止替缺席作证。**

## 禁止的写法
- 禁止再开新案、新阴谋、新母链、新幕后手。  
- 禁止把尾声写成发布会、新闻稿、联席纪要连续播报。  
- 禁止再做一轮“阮见微要公开 / 俞照临要限域 / 邵持衡讲稳定”的完整辩论，只能一句到位。  
- 禁止姐姐再现身解释世界，或留下“其实还有最后真相”的钥匙。  
- 禁止把开放感建立在关键义务不交代上：公众是否接受一次、陆既白是否被承认为现时本人、责任由谁承担，这些都要落下。  
- 禁止高潮后反复落锤：**“留空合法”最多只再验证一次。**

## 下一章质量目标
下一章只做一件事：**让新秩序在普通现实里被真正接住。**  
必须出现一次“留空但有效”的正式办理完成；必须让阮、俞、邵三方责任在同一场景里一句话落位；必须让他人承认陆既白现在的签认资格——他可以守原始链，但不会再代写。  
如果还剩最后一章，那章就只负责余韵：**缩时间、减信息、给画面。** 用一个能呼应开篇意象的安静场景收尾，让读者停在“空白被承认”上，而不是停在“制度已办结”上。
