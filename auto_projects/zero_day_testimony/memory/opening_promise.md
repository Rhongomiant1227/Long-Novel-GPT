# 开篇承诺
## 读者最先被什么抓住
最先抓住读者的，不是单一命案，而是一个极强、极“可推理”的异常：**死者死于地铁坠轨，却没有进站记录；多条证据链各自真实，拼起来却互相否定。** 这一下就把作品和普通刑侦、怪谈区分开了。读者被吸进去的核心快感是：这不是“谁撒谎”，而是“为什么真话会彼此冲突”。与此同时，档案馆、回灌、官方认证链这些冷硬制度细节，又让悬疑不飘，形成一种“系统看似正常，但有一处明显错了”的压迫感。

## 开头提出的核心问题
开头真正抛出的总问题有两层，而且缠得很紧。  
第一层是世界谜题：**第零天是否真实存在？如果存在，谁有权删掉整座城市的一天，并把它改写成所有系统共同承认的现实？**  
第二层是主角命题：**陆既白为何会一边追查，一边反复遗忘；他到底是在逼近真相，还是在逼近一个自己也曾参与过的秘密？**  
姐姐线不是简单“找失踪亲人”，而是更刺的一问：**他想找回的陆南枝，是否早已主动站到了删日内部，并把他预先写进了这场追查里？**

## 核心情绪 / 关系 / 意象
核心情绪承诺是：**冷静推进中的自我侵蚀感。** 主角越校验真相，越失去自身记忆；越想证明现实被篡改，越发现自己也可能是篡改链条的一环。  
核心关系张力，不是单纯敌我，而是“追根”与“止损”、“证据”与“人”的持续冲突，尤其是陆既白与陆南枝、俞照临、连续性局之间的伦理对撞。  
最值得回响的开篇意象有两个：一是**“不存在的到站记录”**，它浓缩了全书的方法论与气质；二是姐姐录音那句——**“说明你又把第零天忘了。”** 这是全书最关键的情感与命运承诺：真相不是他第一次接近，它曾经进入过他的人生，并带走了他的一部分自我。

## 结尾必须兑现什么
真正有力的终局，必须同时兑现四件事：  
1. **讲清删日机制**：不是只说“有阴谋”，而要清楚说明起源、运行逻辑、代价、为何必须靠制度维持。  
2. **讲清姐姐的选择**：陆南枝为什么主动进入/留在删日，她不是受害者标签，而是一个作出残酷且合理选择的人。  
3. **讲清主角与秘密的关系**：他为何早被写进卷宗、为何失忆、为何必须由他来“保留原始记录”。  
4. **让代价落到人物身上**：终局不能只是揭露真相，必须让陆既白在“公开真相”与“维持连续性”之间作出不可逆选择，并真正失去些什么。  
最好通过**一次最终校验/证词重组/公开或封存的决断**来回应，让制度谜底、亲情命运、自我代价在同一动作里闭合。

## 最容易写砸的收尾方式
最容易写砸的，是把结尾写成“案件解释会”或“设定说明书”：把删日原理交代完、把反派动机说圆、把姐姐下落补齐，就算结束。这样会显得虎头蛇尾，因为开篇承诺的不是知识补完，而是**真相对人造成的切割**。  
另外几种危险收尾是：把姐姐降格成线索工具人；把连续性局写成纯恶；让主角几乎无损通关；或靠新设定一把兜底，抹掉前面“证词守恒”和公平推理的魅力。若终局不能让“不存在的到站记录”和“你又把第零天忘了”产生回响，全书最初抓住读者的东西就等于没兑现。
